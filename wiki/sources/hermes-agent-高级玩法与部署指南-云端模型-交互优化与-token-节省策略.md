## 执行摘要 (Executive Summary)

Hermes Agent 作为一款新兴的智能代理工具，其 GitHub 仓库的星标（Star）增长速度已超越 OpenCloud。相较于 OpenCloud，Hermes Agent 以其卓越的稳定性脱颖而出，避免了频繁更新带来的系统崩溃与漏洞。本指南涵盖了三个核心进阶技能：
1.  **零成本准入门槛**：通过 [[Ollama]] 集成的云端免费模型（如 Minimax M2.7），实现不消耗本地资源的极速部署。
2.  **界面交互重塑**：集成 [[Open WebUI]]，解决微信等传统聊天工具在 Markdown 解析和多轮对话管理上的局限性。
3.  **效能与成本平衡**：引入主副模型（Main/Sub Model）分工机制，利用廉价模型处理辅助任务，显著降低 Token 消耗。

## 核心要点

本报告深入探讨了 [[Hermes Agent 高级玩法与部署指南：云端模型、交互优化与 Token 节省策略]] 的关键技术与策略，旨在帮助用户最大化其效能并优化成本。

### 1. Ollama 云端免费模型集成

[[Ollama]] 已原生内置于 Hermes Agent，为用户提供了极低的部署门槛。其技术优势在于用户无需复杂的配置即可运行本地开源模型，或直接调用 Ollama 提供的云端免费额度，例如 Minimax M2.7。这种方式将计算压力完全转移至云端，不占用本地硬件资源，实现了快速且零成本的部署。部署流程简便，只需通过官方网站下载对应操作系统的版本，安装后在终端运行指定命令即可选择并使用模型。

### 2. Open WebUI：提升交互深度

尽管 Hermes Agent 支持连接微信等工具，但在处理复杂格式（如代码块、Markdown）和历史记录回溯方面存在局限。[[Open WebUI]] 的引入彻底解决了这些问题，显著提升了用户交互体验。其功能特性包括：
*   **会话管理**：左侧侧边栏保存完整的历史记录，并支持搜索历史对话，方便用户追溯和管理。
*   **代码增强**：支持 Python 等语言的流式输出与直接运行，例如现场运行冒泡算法，极大地提高了开发和调试效率。
*   **多模态支持**：具备语音输出、修改回复、重新生成以及引用网页/文件/截图的功能，使交互更加丰富和灵活。
*   **跨设备访问**：用户可以通过电脑局域网 IP 地址及端口（8080），在手机浏览器上获得完整的 Agent 交互体验，实现无缝切换。

### 3. 主副模型配置优化策略

针对用户普遍反映的 Agent 运行成本高（“烧 Token”）的问题，Hermes Agent 提供了细粒度的模型指派功能。通过引入主副模型分工机制，用户可以为不同类型的任务配置最合适的模型，从而显著降低 Token 消耗。

| 任务类型 | 推荐配置策略 |
| :--- | :--- |
| **主任务** | 处理复杂逻辑、核心分析，建议使用高性能模型以确保准确性和效率。 |
| **辅助任务** | 处理批准（Approval）、压缩（Compression）、记忆刷新、MCP 调用、视觉分析、网页搜索等。 |
| **优化方案** | 为辅助任务指定更便宜的模型，如 Google Gemini 1.5/2.5 Flash，以实现成本效益最大化。 |

### 5. 技术配置与行动指南

#### 5.1 Open WebUI 的连接与配置

若要在 Open WebUI 中使用 Hermes Agent，需执行以下步骤：

1.  **启用 API 服务**：
    修改 `agent` 配置文件，添加两个关键参数：
    *   `--api`: 启用 API 服务。
    *   `--api-key [自定义密码]`: 设置访问权限。
    *(提示：可使用 Codex 或 Claude 代码助手自动执行此配置)*
2.  **网关重启**：执行 `gateway` 重启命令以应用更改。
3.  **UI 连接设置**：
    *   在 Open WebUI 管理员设置中添加连接。
    *   URL 填写：`http://localhost:8642/v1`（兼容 OpenAPI 接口）。
    *   输入预设的 API 密钥并保存。

#### 5.2 节省 Token 的配置文件示例

在配置文件中，可针对不同子任务细化模型 ID 和提供商。建议将以下辅助任务统一指向轻量化模型：

*   **配置项包含**：`approval`（批准）、`compression`（压缩）、`session_search`（搜索）、`vision`（视觉）、`web_search`（网页搜索）。
*   **推荐操作**：删除默认冗余配置，替换为 Gemini Flash 或类似的低成本 API 接口。

#### 5.3 手机端远程交互

*   **访问方式**：在同一局域网下，通过 `http://[电脑IP]:8080` 登录 Open WebUI 账号。
*   **进阶需求**：若需在公网环境下访问，建议配合内网穿透工具使用。
