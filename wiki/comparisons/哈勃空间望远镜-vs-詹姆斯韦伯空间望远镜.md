# 哈勃空间望远镜 vs 詹姆斯·韦伯空间望远镜

## 对比维度表格

| 维度 | 哈勃空间望远镜 (HST) | 詹姆斯·韦伯空间望远镜 (JWST) |
| :--- | :--- | :--- |
| **主要波段** | 可见光、紫外线 | 近红外、中红外 |
| **主镜直径** | 2.4 米 | 6.5 米 |
| **工作温度** | 约 20°C（近地轨道） | 低于 -223°C（L2点） |
| **科学核心** | 现代星系演化、暗能量、系外行星 | 宇宙黎明、恒星诞生、系外行星大气探测 |
| **部署位置** | 近地轨道（约540 km） | 日地拉格朗日L2点（约150万 km） |
| **可维护性** | 航天飞机维修（已终止） | 不可维护 |
| **运营时间** | 1990年至今（34年） | 2021年至今（5年） |
| **代表性图像** | 哈勃深场、创生之柱（可见光） | SMACS 0723、创生之柱（红外） |

## 核心差异分析
1. **波段互补**：哈勃的可见光/紫外能力与韦伯的红外能力形成完美互补，哈勃适合观测热气体、年轻恒星和星系核，韦伯适合观测冷尘埃、高红移宇宙和原恒星。
2. **观测深度与分辨率**：韦伯的6.5米主镜使其在红外波段具有比哈勃更高的空间分辨率（衍射极限），同时集光面积是哈勃的6倍，能探测更暗弱的天体。然而在可见光波段哈勃仍有优势。
3. **热环境影响**：哈勃在近地轨道经历日夜交替，温度波动大；韦伯在L2点背向太阳，采用多层遮阳罩稳定冷却，适合极灵敏的红外探测。
4. **科学与公众影响**：哈勃定义了现代天文学公共形象，而韦伯正在改写早期宇宙和系外行星领域。

## 适用场景结论
- **需要紫外/可见光高分辨率成像**（如研究星系形态、超新星遗迹细节）：选择[[哈勃空间望远镜]]。
- **需要穿透尘埃探测恒星形成核心**（如观察原恒星喷流、星周盘）：选择[[James-Webb空间望远镜]]。
- **研究宇宙早期（z>6）星系和第一代恒星**：必须依靠韦伯的红外能力。
- **系外行星大气化学表征**：韦伯的中红外透射光谱优于哈勃，但哈勃的可见光/近红外光谱在部分指标（如水、钠）仍有应用。
- **最佳实践**：两者联合观测，如对相同天区进行多波段成像（X射线+可见光+红外），可获得最完整的物理信息。
