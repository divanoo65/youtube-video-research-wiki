# 零月费模式

## 定义

零月费模式指通过使用开源大语言模型（如 Qwen3.6）和自托管智能体（如 Hermes Agent），在本地硬件上运行 AI 服务，无需向任何云服务商支付订阅费用，实现真正的“Token 自由”。

## 在本库的具体例子

在 [[Hermes-Qwen3-6-本地-AI-Agent-组合部署与应用简报]] 中，该方案被描述为“完全免费，真的可以做到 TOKEN 自由”，与 ChatGPT Plus 等付费服务形成对比。

## 技术实现细节

用户需要一次性投入硬件成本（如带 N 卡的个人电脑），然后通过开源软件栈（WSL、Llama-cpp、Hermes Agent、Qwen 模型）搭建服务。后续使用中，仅需支付电费和网络费用，无额外月费。

## 与近似概念的边界

- **免费 API 额度**：免费 API 额度有限制，零月费模式无限制。
- **订阅制**：订阅制按月付费，零月费模式一次性硬件投入。
