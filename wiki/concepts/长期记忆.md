# 长期记忆

## 定义

长期记忆指大语言模型能够参考过去的聊天记录和关联数据，在后续对话中提供个性化、上下文相关的回应。GPT-5.5 Instant 通过“Memory Sources”功能实现，能结合用户的地理位置、内容偏好等信息进行建议。

## 在本库的具体例子

在 [[GPT-5-5-Instant-深度分析简报-核心特性与实测表现]] 中，模型能记住用户位于旧金山，并在推荐餐厅时结合当地信息，体现了地域与偏好感知。

## 技术实现细节

Memory Sources 功能通过提取历史对话中的关键信息（如用户姓名、地点、偏好），存储为结构化记忆，在每次对话启动时加载到上下文中。模型会主动判断哪些记忆与当前问题相关，并加以利用。

## 与近似概念的边界

- **上下文窗口**：上下文窗口是单次对话内的短期记忆，长期记忆跨越多次会话。
- **程序化知识生成**：程序化知识生成是 Hermes Agent 的技能机制，侧重于记住方法而非事实。
