# 云端免费模型

## 定义
云端免费模型是指通过 [[ollama]] 平台使用的、运行在云端服务器上的 AI 模型，无需占用用户本地硬件资源。在 [[hermes-agent]] 的部署场景中，用户可以通过 Ollama 选择带“Cloud”标识的模型（如 Minimax M2.7）来使用免费云端算力。

## 在本库的具体例子

- 在 [[hermes-agent-高级玩法与优化指南-深度简报]] 中，视频介绍了如何通过 Ollama 使用云端免费模型 Minimax M2.7，无需占用本地硬件资源。
- 具体操作：在 Ollama 中运行内置命令，选择带“Cloud”标识的模型即可使用免费云端算力。

## 技术实现细节

1. **Ollama 集成**: Ollama 已原生集成 Hermes Agent，用户只需选择带“Cloud”标识的模型即可使用云端算力。
2. **无需本地资源**: 模型运行在云端服务器，不占用用户本地 CPU/GPU 资源。
3. **傻瓜化配置**: 整个配置过程高度集成，支持一键式部署。

## 与近似概念的边界

- **本地模型**: 本地模型运行在用户自己的硬件上，需要占用本地资源，而云端免费模型运行在云端服务器上。
- **付费 API 模型**: 付费 API 模型（如 OpenAI GPT-4）需要按使用量付费，而云端免费模型完全免费。
