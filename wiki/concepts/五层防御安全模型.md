# 五层防御安全模型

## 定义

五层防御安全模型是 Hermes Agent 为安全部署在暴露互联网环境而设计的纵深防御体系，包括用户授权、危险命令审批、容器隔离、MCP 凭证过滤、上下文文件扫描五层，并辅以 SSRF 防护和网站黑名单。

## 在本库的具体例子

在 [[Hermes-Agent-与-OpenClaw-深度对比简报]] 中，该模型被描述为 Hermes Agent 的核心安全特性，使其能安全地暴露在公网。

## 技术实现细节

- **用户授权**：验证操作者身份，防止未授权访问。
- **危险命令审批**：对高风险命令（如删除文件）进行二次确认。
- **容器隔离**：在 Docker 容器中运行任务，限制系统访问。
- **MCP 凭证过滤**：防止敏感凭证泄露。
- **上下文文件扫描**：检查输入文件是否包含恶意内容。

## 与近似概念的边界

- **沙箱**：沙箱是单一隔离机制，五层防御是组合策略。
- **权限管理**：权限管理是用户授权的一部分，五层防御涵盖更广。
