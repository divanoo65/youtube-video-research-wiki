## 定义

分层记忆体系是 Hermes Agent 构建的精密的按需检索记忆系统，包含四个层次：1）核心持久记忆（MEMORY.md 和 USER.md，严格限制 ~1300 tokens），2）会话历史记忆（基于 SQLite 的 state.db，通过 FTS5 全文索引实现跨会话精准检索），3）扩展记忆层（Honcho 长期用户画像），4）程序性记忆（存储「如何做」的操作模式）。相比 OpenClaw 以 Markdown 文件为核心的动态性较弱的记忆模式，Hermes 的记忆体系实现了质的飞跃。

## 在本库的具体例子

- Hermes Agent 的 `session_search` 工具基于 FTS5 实现跨会话精准检索（参见 [[Hermes Agent 与 OpenClaw 智能体架构深度对比]]）
- `skill_manage` 工具作为程序性记忆层，存储可复用的技能和工作流

## 关联概念

- [[程序化知识]]：分层记忆体系中的程序性记忆层
- [[Agent 执行循环]]：执行循环依赖分层记忆提供上下文的持续优化
