# 分层记忆体系

## 定义

一种结构化智能体记忆管理系统，将记忆按持久性和抽象程度划分为不同层级。[[hermes-agent]] 实现了一个四层记忆体系：L1 核心持久层（MEMORY/USER.md，限额 1300 tokens）、L2 会话历史层（SQLite+FTS5 全文检索）、L3 扩展长期层（Honcho 模块构建长期用户画像）、L4 程序化记忆层（自动生成的技能库）。

## 本库具体例子

在 [[Hermes Agent 与 OpenClaw 深度对比及技术趋势简报]] 中，Hermes Agent 的四层记忆体系被表格化对比：

| 层级 | Hermes Agent | OpenClaw（对比） |
|------|-------------|-----------------|
| L1 核心持久 | MEMORY/USER.md，1300 tokens 限额 | Markdown 文件 |
| L2 会话历史 | SQLite + FTS5 全文索引 | 文件系统持久化 |
| L3 扩展长期 | Honcho 模块，用户画像 | 工作区身份配置 |
| L4 程序化记忆 | 自动生成的技能库 | 预设技能插件包 |

## 与其他概念的关系

- L4 程序化记忆层即[[程序化知识生成]]的体现
- 分层架构是[[去中心化执行循环]]的重要支撑
- 分层记忆比传统单层记忆更适合长时序任务

## 关联概念

- [[程序化知识生成]]：L4 记忆层的核心机制
