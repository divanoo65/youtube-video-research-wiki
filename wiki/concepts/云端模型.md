## 概念定义

云端模型是指部署在远程服务器或云计算基础设施上，并通过网络提供服务的人工智能模型。与传统的本地部署模型不同，用户无需在本地设备上安装和运行复杂的AI模型，而是通过互联网访问和调用云端提供的计算资源和模型服务。这种模式极大地降低了用户对本地硬件性能的要求，并允许用户按需使用、按量付费，甚至在某些情况下（如通过Ollama集成的免费额度）实现零成本使用。在[[Hermes Agent 高级玩法与优化深度简报]]中，云端模型特指通过[[Ollama]]框架集成的、可利用云端免费额度运行的AI模型，例如[[MX M2.7]]及其云端版本，为[[AI Agent 框架]]如[[Hermes Agent]]提供了便捷、高效且经济的部署和运行方案。

## 技术细节

云端模型的核心技术在于将AI模型的计算和存储任务转移到远程的云计算平台。对于[[Hermes Agent]]而言，其与[[Ollama]]框架的集成是实现云端模型利用的关键。

1.  **集成与部署:** [[Hermes Agent]]现已内置于[[Ollama]]框架中。用户只需通过简单的命令行指令，即可完成赫美斯的配置与运行，实现了[[一键化部署]]。
2.  **资源利用:** 通过[[Ollama]]提供的云端模型（如[[MX M2.7]]及其云端版本），用户可以在不占用本地CPU、GPU等硬件资源的情况下，利用云服务商提供的免费额度或按需付费资源来运行[[AI Agent]]。这属于典型的[[云端资源利用]]模式。
3.  **操作流程:** 典型的操作流程包括：下载并安装[[Ollama]] -> 运行指定命令以拉取并启动赫美斯模型 -> 在浏览器中登录账号并连接设备 -> 在终端激活[[赫美斯网关]]。整个过程强调了便捷性和对云端资源的依赖。
4.  **成本效益:** 这种模式显著降低了AI Agent的运行成本，尤其对于个人开发者和小型团队，通过利用云端免费额度，可以实现“零成本部署”。

## 应用场景

云端模型在[[AI Agent 框架]]的部署和运行中具有广泛的应用，特别是在以下场景中展现出其优势：

1.  **零成本或低成本部署:** 对于希望尝试或运行[[AI Agent]]但缺乏高性能本地硬件的用户，云端模型提供了一个经济高效的解决方案。通过利用[[Ollama]]等平台提供的免费额度，可以实现[[Hermes Agent]]的零成本部署。
2.  **便捷性与可访问性:** 用户无需进行复杂的本地环境配置，只需简单的几步操作即可通过网络访问和使用AI模型，极大地降低了技术门槛。
3.  **资源弹性与扩展性:** 云端模型可以根据需求动态调整计算资源，无论是处理少量请求还是应对高并发场景，都能提供灵活的扩展能力。
4.  **跨平台与设备无关:** 用户可以通过任何连接互联网的设备（如电脑、平板、手机）访问和控制云端运行的[[AI Agent]]，实现真正的设备无关性。
5.  **教育与研究:** 为学生和研究人员提供了一个无需投入大量硬件成本即可进行AI模型实验和开发的平台。
