# 五层纵深防御

## 定义

五层纵深防御是 [[hermes-agent]] 针对本地智能体权限过大风险设计的默认开启安全模型，包含核心防御层和额外保护层，覆盖从用户交互到网络访问的全链路安全。

## 五层核心防御

1. **用户授权：** 用户确认后方可执行操作
2. **危险命令审批：** 高风险命令需用户单独批准
3. **容器隔离：** 在隔离环境中运行不信任的操作
4. **MCP 凭证过滤：** 过滤敏感的 MCP 凭证信息
5. **上下文文件扫描：** 扫描上下文中的潜在风险文件

**额外保护层：** SSRF 防护、网站黑名单、环境变量过滤、危险命令预执行扫描。

## 在本库的具体例子

- [[Hermes-Agent-与-OpenClaw-深度对比简报]] 中描述了 Hermes Agent 的五层纵深防御模型。默认开启的安全机制确保 Agent 在拥有强大工具执行能力的同时，不会对用户的系统、数据和网络造成安全威胁。
