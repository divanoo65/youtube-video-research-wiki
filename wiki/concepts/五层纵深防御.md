# 五层纵深防御

## 定义

[[hermes-agent]] 默认配置的多层安全防御模型，从用户交互到底层执行环境逐层加固。五层包括：用户授权（交互确认）、危险命令审批（自动扫描敏感操作）、容器隔离（执行环境沙箱化）、MCP 凭证过滤（保护第三方服务凭证）、上下文文件扫描（检测恶意文件）。额外包含 SSRF 防护和网站黑名单机制。

## 本库具体例子

在 [[Hermes Agent 与 OpenClaw 深度对比及技术趋势简报]] 中，五层防御被视为 Hermes Agent 区别于 OpenClaw 的安全优势。这是针对 OpenClaw 早期安全争议的针对性加固，使得 Hermes Agent 更适合无人值守的自动化场景（如 Cron 定时任务）。

## 与其他概念的关系

- 五层防御使[[去中心化执行循环]]能在无人值守环境下安全运行
- 内置 Cron 系统的安全运行依赖此防御模型
- 与 [[openclaw]] 相比，Hermes Agent 在安全性上进行了主动加固
