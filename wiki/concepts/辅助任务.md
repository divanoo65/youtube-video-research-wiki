# 辅助任务

## 定义
辅助任务是 AI Agent 运行过程中需要处理的、不依赖深度推理的轻量级操作。在 [[Hermes-Agent-赫美斯]] 的主副模型架构中，辅助任务被统一分配给低成本副模型（如 [[Gemini-1.5-Flash]]）处理，而主模型则专注于需要高推理能力的核心任务。

## 辅助任务分类
在 Hermes Agent 中，辅助任务包括但不限于：
- **任务批准（Approval）**：对 Agent 即将执行的操作进行审批确认
- **上下文压缩（Compression）**：对长对话历史进行压缩存储
- **记忆刷新（Memory Refresh）**：恢复和更新压缩后的历史信息
- **MCP 调用**：执行 MCP（Model Context Protocol）工具的调用
- **会话搜索（Session Search）**：在历史会话中搜索相关信息
- **视觉处理**：对图像进行识别和分析
- **网页工具**：执行网页浏览、内容提取等操作

## 技术实现细节
在 Hermes Agent 的配置文件中，辅助任务通过 `auxiliary_tasks` 字段配置。每个辅助任务类型可以单独指定模型，最佳实践是统一指向同一个低成本模型。
