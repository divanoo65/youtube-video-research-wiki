## 定义

五层纵深防御模型是 Hermes Agent 为应对本地智能体运行高权限风险而设计的默认安全防护体系。五层防御包括：用户授权（第一层）、危险命令审批（第二层）、容器隔离（第三层）、MCP 凭证过滤（第四层）、上下文文件扫描（第五层）。此外还包含 SSRF 防护、网站黑名单、环境变量过滤、消息用户配对等额外机制。

## 在本库的具体例子

- Hermes Agent 在执行高危操作（如 Shell 命令执行）时触发审批流程（参见 [[Hermes Agent 与 OpenClaw 智能体架构深度对比]]）
- 容器隔离确保即使 Agent 被攻破，宿主系统仍然安全

## 关联概念

- [[Agent 执行循环]]：执行循环在高权限场景下与安全防御模型协同工作
- [[程序化知识]]：技能自动生成同样受到安全防御模型的约束

## 关联实体

- [[Hermes Agent]]：五层纵深防御模型是其安全体系的核心
- [[OpenClaw]]：未提及类似层级化安全防御机制
