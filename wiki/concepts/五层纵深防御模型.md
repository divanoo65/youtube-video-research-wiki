# 五层纵深防御模型

## 定义
Hermes Agent 针对自托管 Agent 权限过大的安全风险提出的多层次纵深防御体系。由外到内依次为用户授权、危险命令审批、容器隔离、MCP 凭证过滤和上下文扫描，额外增加 SSRF 和恶意注入防护。

## 本库的具体例子

在 Hermes Agent 中，五层防御的具体实现：

1. **用户授权**: 所有操作需用户明确授权
2. **危险命令审批**: 对高风险 Shell 命令进行二次确认审批
3. **容器隔离**: 关键任务在隔离环境中运行，防止横向移动
4. **MCP 凭证过滤**: 对 Model Context Protocol 连接的凭证进行精细化过滤
5. **上下文扫描**: 检测并阻断恶意注入攻击

此外，在 Hermes + Qwen3.6 部署指南中特别提到：Telegram Bot 权限应设置为"个人使用"并绑定特定 Telegram ID，这是用户授权层的具体实践。
