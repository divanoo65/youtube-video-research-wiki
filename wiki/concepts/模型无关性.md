# 模型无关性

## 定义

模型无关性指 Hermes Agent 通过 `hermes model` 命令即可切换 OpenAI、OpenRouter 以及各类国产大模型服务，无需修改核心代码或重新配置。这种设计使用户可以灵活选择最适合当前任务的后端模型。

## 在本库的具体例子

在 [[Hermes-Agent-与-OpenClaw-深度对比简报]] 中，模型无关性被列为 Hermes Agent 的核心特性之一，与 OpenClaw 的固定模型绑定形成对比。

## 技术实现细节

Hermes Agent 抽象出一个统一的 API 接口层，所有后端模型都通过该接口通信。`hermes model` 命令本质上是切换配置文件中的模型端点、API Key 和参数模板，无需重启服务。

## 与近似概念的边界

- **chat-latest API 策略**：chat-latest 是 OpenAI 的特定策略，模型无关性是 Hermes 的通用能力。
- **多模型支持**：多模型支持指同时使用多个模型，模型无关性指切换的便捷性。
