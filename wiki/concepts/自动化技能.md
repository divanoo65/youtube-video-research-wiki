# 自动化技能

## 定义

自动化技能（Automated Skills）是指智能体在执行任务过程中自动将成功的多步骤操作模式抽象为可复用的 skill 文件并存入技能库，从而在下一次遇到类似任务时可以直接调用，无需重新编写或手动配置。

## 在本库的具体例子

- **Hermes Agent 的 Skill 系统**：Agent 自动将有效的多步操作模式保存为 skill 文件存于 `~/.hermes/skills/` 目录。下次遇到类似任务时自动加载对应的 skill。
- 与 OpenClaw 的对比：OpenClaw 的技能主要由人类开发者以插件扩展包形式编写，而 Hermes 的技能是自动生成和自我更新的。
