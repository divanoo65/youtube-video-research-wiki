# 远程调用

## 定义

远程调用指用户通过 Telegram Bot 等消息平台，在手机端发送指令，远程控制部署在本地电脑上的 Hermes Agent 执行任务，包括对话、编程、定时任务等。这使得本地静态模型转变为全球可达的动态自动化工具。

## 在本库的具体例子

在 [[Hermes-Qwen3-6-本地-AI-Agent-组合部署与应用简报]] 中，通过配置 Telegram Bot Token 并绑定个人 ID，用户可以在任何地方通过手机调用家里的本地模型执行定时任务。

## 技术实现细节

Hermes Agent 作为后台服务运行，监听 Telegram Bot 的 Webhook 或长轮询。收到用户消息后，Agent 调用本地模型处理，并将结果通过 Bot 回复给用户。整个过程在本地完成推理，仅传输文本结果。

## 与近似概念的边界

- **SSH 远程**：SSH 需要命令行操作，远程调用通过自然语言交互。
- **API 调用**：API 调用需要编程，远程调用通过聊天界面。
