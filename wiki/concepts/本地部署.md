# 本地部署

## 定义

本地部署是指在个人计算设备（如 PC、工作站、服务器）上直接安装和运行 AI 模型及智能体系统，所有数据处理和推理在本地完成，无需依赖云服务或 API 调用。

## 在本库的具体例子

- **Qwen3.6 + Hermes Agent**：在 Windows WSL2 环境中通过 Llama-cpp 运行 Qwen3.6 模型（从 0.8B 到 27B），结合 Hermes Agent 构建完整 AI 助手系统。
- **Hermes Agent 多环境支持**：支持本地、VPS、Docker、SSH 远程环境及无服务器架构，部署灵活。
- **开机自启**：配置文件自启动脚本，成为 24 小时在线的本地服务端。
