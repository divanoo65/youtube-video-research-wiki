# 本地部署

## 定义
在个人或企业自有硬件上完整部署和运行 AI 模型及其配套 Agent 框架，所有计算和数据均在本地完成，不依赖第三方云端 API 服务。

## 本库的具体例子

Hermes + Qwen3.6 方案提供了一个完整的本地部署范例：
- **底层环境**: Windows + WSL Ubuntu 24.04
- **推理加速**: Llama-cpp + CUDA，保持显存调度稳定性
- **模型选择**: Qwen3.6 系列（0.8B~27B 根据显存选择）
- **应用层**: Hermes Agent，通过 Telegram Bot 实现远程交互
- 所有对话数据存储在本地 Ubuntu 子系统中，确保隐私安全
- 通过 Telegram Bot 实现的手机远程调用仍然是本地执行——指令经 Telegram 通道下发到本地模型处理
