## 定义

程序化知识（Procedural Knowledge）是 Hermes Agent 独有的知识形态——技能不再需要人工编写，而是由 Agent 根据执行经验自动生成。这标志着智能体从传统的「记住事实」升级为「记住方法」，是从被动执行到主动成长的关键跨越。Hermes Agent 将技能理解为「可复用的程序化流程」，通过 `skill_manage` 工具自主完成创建、优化与管理。

## 在本库的具体例子

- Hermes Agent 能根据执行经验自动创建新技能，无需人工编写，遵循 agentskills.io 开放标准（参见 [[Hermes Agent 与 OpenClaw 智能体架构深度对比]]）
- 相比 OpenClaw 需要人类开发者编写模块化插件，Hermes Agent 的技能机制实现了质的飞跃
