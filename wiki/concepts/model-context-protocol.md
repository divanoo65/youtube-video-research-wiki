# MCP (Model Context Protocol)

## 概念定义

MCP（Model Context Protocol，模型上下文协议）是一种新兴的开放协议标准，旨在为 AI 模型（尤其是大语言模型）提供一个统一、安全且可扩展的接口，使其能够与外部世界进行交互。在传统 AI 应用中，模型通常只能处理静态的输入文本或知识库，而无法直接调用实时数据、操作外部工具或访问私有数据库。MCP 的出现打破了这一限制：它定义了模型与“工具服务器”（Tool Server）之间的通信规范，包括身份认证、权限控制、工具发现、参数传递与结果返回等机制。通过 MCP，AI 智能体（如 [[hermes-agent-self-evolving-ai-agent]] 中描述的自进化 Agent）可以无缝地接入浏览器、文件系统、数据库、网页 API、电子邮件等各种资源，而无需为每个集成编写自定义代码。MCP 的核心价值在于“标准化”——它将碎片化的工具调用协议统一为一个通用层，使得模型只需要理解一种协议就能访问多种服务，极大降低了智能体的开发与维护成本。同时，MCP 内置了安全沙箱与权限管理，防止模型误操作或越权访问，从而保障了在实际部署中的可控性。从架构上看，MCP 遵循客户端-服务器模式，AI 模型作为 MCP 客户端，通过 JSON-RPC 协议与运行在宿主环境中的 MCP 服务器进行双向通信，服务器负责抽象底层资源的差异并提供统一的接口描述。这种设计使得模型可以“动态发现”可用工具，并像使用内置函数一样调用它们，从而真正实现从“对话式”到“行动式”的跨越。

## 技术细节

MCP 协议的技术核心包括以下几个关键部分：

- **协议层**：基于 JSON-RPC 2.0 标准，使用 WebSocket 或标准 HTTP/2 作为传输层，支持双向流式通信。请求与响应均采用结构化的 JSON 格式，包含方法名、参数和唯一 ID。
- **工具发现（Tool Discovery）**：MCP 服务器会向客户端（AI 模型）暴露一个“工具清单”（list of tools），每个工具由名称、描述、输入参数模式（JSON Schema）以及输出格式定义。模型可以在运行时通过 `listTools` 方法获取所有可用工具，并在后续调用中按需选择。
- **安全与权限**：MCP 服务器可定义精细的权限策略，例如只允许读取特定文件、限制网络访问范围、设置调用频率等。模型发出的每次工具调用都会经过服务器端的权限校验，避免模型自主越权。
- **上下文管理**：MCP 支持长上下文场景，服务器可以维护会话状态，并在多次调用之间共享上下文（如文件句柄、数据库连接等）。这使得模型能够执行多步骤的复杂任务，例如逐步收集信息、修改配置后再次查询结果。
- **错误处理与回退**：当工具调用失败（如网络超时、权限不足、参数错误）时，MCP 会返回结构化的错误码和错误信息，模型可以据此决定重试、回退或向用户请求补充信息。

上述技术细节与 [[learning-cycle]] 密切相关——Agent 通过 MCP 调用外部工具获取反馈，并将结果纳入自身记忆中，从而驱动后续的自改进与技能演化。

## 应用场景

MCP 极大地拓展了 AI 智能体的能力边界，典型的应用场景包括：

1. **自动化办公与数据处理**：Agent 通过 MCP 连接到公司的 CRM 系统、电子邮件客户端、日历应用，自动完成客户信息查询、会议安排、报告生成等任务。Agent 不再只是“回答问题”，而是“执行操作”。
2. **代码开发与运维**：Agent 通过 MCP 访问代码仓库（GitHub、GitLab）、CI/CD 流水线、云服务器控制台、数据库管理员接口，实现代码审查、自动部署、故障排查等。在 [[self-evolution]] 的 Agent 框架中，Agent 甚至可以编写脚本并通过 MCP 执行，然后观察结果来改进自身策略。
3. **个人助理与生活服务**：Agent 通过 MCP 连接天气 API、地图服务、购物平台、家庭智能设备，为用户提供一站式的生活决策支持。例如，用户说“帮我安排本周的晚餐计划”，Agent 可以自动查询天气、搜索菜谱、下单生鲜，并设置厨房提醒。
4. **企业级知识管理**：Agent 通过 MCP 接入内部的知识库、Wiki、文档系统，在回答用户问题时自动检索最新资料，并将权威信息源返回给用户，减少幻觉风险。结合 [[user-modeling]]，MCP 还能根据用户的权限范围自动过滤结果。

MCP 正在成为 AI 智能体领域事实上的“连接层”标准，与 [[openai]] 等模型提供的原生函数调用相比，它更强调开放性与跨平台互操作性，使得不同厂商的模型和工具可以即插即用。未来，随着更多第三方服务提供 MCP 服务器，AI 智能体的能力将进一步爆发。
