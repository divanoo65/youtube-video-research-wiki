# 零成本部署

## 定义

通过本地开源模型和自托管智能体框架实现 AI 部署的模式，不产生按量计费的云端 API 费用，数据无需上传，完全私有化运行。核心是 WSL + CUDA + Llama-cpp + 开源模型 + Hermes Agent 的技术栈组合。

## 本库具体例子

在 [[Hermes + Qwen-3.6 本地最强 Agent 组合部署与应用简报]] 中，[[hermes-agent]] 与 [[qwen-3-6]]（27B）组合在 24GB 显存环境下实现了 40+ tokens/s 的推理速度，支持 Telegram 远程交互和定时任务。用户只需一次性的硬件投入，无需月度订阅费用。

关键引文："普通人很多的任务都不需要用收费模型，本地模型已经足够使用了……关键是它是完全免费的，无须任何费用，从此不再受限。"

## 与其他概念的关系

- 零成本是[[Token自由]]的具体体现
- 零成本部署依赖[[本地私有化]]实现数据安全
- 与云端付费方案（如 [[gpt-5-5-instant]] 的 API 费用）形成对比
