# 程序化知识生成

## 定义

程序化知识生成是指 AI Agent 能够根据自身的执行经验，自动将已完成的工作流抽象为可复用的技能（Skill），而非依赖人类开发者手动编写。这是 Hermes Agent 区别于传统 Agent 框架的核心能力，标志着智能体从被动执行到主动成长的关键跨越。

## 在本库的具体例子

在 [[Hermes-Agent-与-OpenClaw-深度对比简报]] 中，Hermes Agent 内置 `skill_manage` 工具，支持技能的自动创建、优化与管理。例如，当 Agent 成功完成某一类型的任务后，可以自动提取该任务的执行模式，创建一个新的 Skill 以便将来重复使用，并支持 `agentskills.io` 开放标准的 Skills Hub 扩展。

## 技术实现细节

- 使用 `skill_manage` 工具完成技能的创建、更新和删除（全生命周期管理）
- 技能存储在 `~/.hermes/skills/` 目录下，以 YAML frontmatter + Markdown 格式
- 技能可包含触发条件、步骤、代码示例和验证方法
- 支持从 Skills Hub（`agentskills.io` 标准）安装社区共享的技能

## 与近似概念的边界

- **程序化知识生成 ≠ 人工编写技能**：前者由 Agent 自动从经验中提炼，后者由人类手动编写。Hermes 支持两者并存。
- **程序化知识生成 ≠ 函数调用**：函数调用是固定功能的调用，程序化知识是可进化、可分享的工作流模板。
