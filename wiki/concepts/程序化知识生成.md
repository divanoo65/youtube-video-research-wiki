# 程序化知识生成

## 定义

程序化知识生成是 Hermes Agent 的核心能力，指智能体在完成任务后，自动将执行过程中的有效操作序列抽象为可复用的技能文件（存储于 `~/.hermes/skills/`），实现跨会话的持续学习。它让智能体从“记住事实”升级为“记住方法”。

## 在本库的具体例子

在 [[Hermes-Agent-与-OpenClaw-深度对比简报]] 中，Hermes Agent 通过 `skill_manage` 工具管理自动生成的技能，而 OpenClaw 需要人类编写模块化插件。

## 技术实现细节

当 Hermes Agent 完成一个任务后，系统会分析操作轨迹，提取出通用的步骤和参数，生成一个 Markdown 格式的技能文件。该文件包含触发条件、执行步骤和预期结果。后续遇到类似任务时，模型会自动调用匹配的技能，无需从头推理。

## 与近似概念的边界

- **长期记忆**：长期记忆记住事实（如用户偏好），程序化知识生成记住方法（如操作流程）。
- **插件系统**：插件由人类编写，技能由 AI 自动生成。
