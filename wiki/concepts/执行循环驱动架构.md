## 定义

执行循环驱动架构是 Hermes Agent 采用的底层控制中枢设计哲学。它将 AI Agent 自身的执行循环定义为同步编排引擎，网关、工具运行时、通信协议（ACP）和持久化模块均围绕此循环集成。这种设计允许智能体在每一次执行中通过闭环反馈优化自身的决策逻辑。

## 本库具体例子

- 与 OpenClaw 的中心化网关模式形成鲜明对比：OpenClaw 以 Gateway 为绝对控制中枢，而 Hermes Agent 通过执行循环实现自我优化
- 每一次 Agent 执行完毕，其执行轨迹可被自动提取为可复用的 Skill（程序性记忆），这是闭环反馈的具体实现
