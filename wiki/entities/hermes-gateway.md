赫美斯网关是[[Hermes Agent 高级玩法与优化深度简报]]中详细阐述的一个关键组成部分，其核心目标是极大地降低AI Agent的使用门槛并优化其运行效率。它通过提供[[一键化部署]]能力，用户仅需一条简单的命令行指令即可完成配置与运行，极大地简化了Agent的启动流程。在资源利用方面，赫美斯网关支持[[云端资源利用]]，能够通过Ollama等平台提供的[[云端模型]]（例如[[MX M2.7]]及其云端版本），让用户在不占用本地硬件资源的情况下，利用云端的免费额度运行复杂的Agent任务。

为了提升用户交互体验，赫美斯网关原生支持Open WebUI集成，解决了传统聊天工具（如[[微信]]）在Markdown解析、对话历史保存和交互便捷性上的局限。Open WebUI提供了完美的Markdown解析、代码块运行、历史记录搜索、流式输出以及发音、修改、重试、一键复制等功能按钮，并支持移动端访问和多模态输入（文件、截图、网页引用、知识库）。

此外，针对Agent运行成本较高的痛点，赫美斯网关引入了创新的主副模型（Primary/Sub-model）协同机制。该策略将复杂任务分配给昂贵的高性能主模型处理，而将简单辅助任务（如审批、压缩、记忆刷新、MCP调用、视觉分析、网页工具等）交由低廉甚至免费的副模型执行，从而显著优化了[[Token 消耗优化]]，有效降低了整体运行成本。

### 在本报告中的角色
在本报告中，赫美斯网关被介绍为[[Hermes Agent 高级玩法与优化深度简报]]的核心组件之一，详细阐述了其在简化部署、利用云端资源、增强用户交互体验以及通过主副模型策略优化运行成本等方面的关键作用和实现方式。它是实现[[AI Agent 框架]]高效、便捷运行的重要基础设施。
