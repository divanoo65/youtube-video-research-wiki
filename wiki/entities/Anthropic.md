# Anthropic

## 基本定位
美国人工智能公司，以 AI 安全研究闻名，提出 Context Reflected 和独立评估者模式等 Harness 实践。

## 核心特征/能力
1. **AI 安全研究**：专注于 AI 安全性、可控性和伦理问题。
2. **Context Reflected 技术**：通过上下文反射机制优化模型输出质量。
3. **独立评估者模式**：将 Planner、Generator、Evaluator 角色分离，由独立评估者验证输出质量以降低幻觉风险。
4. **Harness Engineering 实践**：提出 Agent = Model + Harness 的核心公式，提供六层成熟架构及行业最佳实践。
5. **Managed Agents 架构**：通过解耦大脑与双手，实现 Agent 的无状态化和高并发处理能力。

## 应用场景
1. **企业级 AI 工具开发**：为大型企业提供安全、可控的 AI 解决方案。
2. **Agent 系统设计**：帮助开发者构建稳定、可扩展的 AI Agent 系统。
3. **AI 安全测试**：提供 AI 安全性评估和测试服务。
