# Ollama

Ollama 是当前 AI 领域广泛应用的大模型运行与管理平台，旨在为用户提供轻量级、高性能的模型部署体验。作为 [[Hermes Agent]] 的核心运行环境，Ollama 通过其高度集成的架构，极大地降低了 AI Agent 的使用门槛。它不仅支持本地模型的快速加载与运行，还通过云端接口的集成，允许用户在不占用本地计算资源的前提下，调用如 MXM 2.7 等高性能云端模型。

在实际应用中，Ollama 实现了“傻瓜化”的配置流程，用户仅需通过简单的安装与终端指令，即可完成环境搭建并激活 Agent 的 Gateway。这种设计不仅保证了系统运行的稳定性，还通过云端与本地的灵活切换，为用户提供了极具性价比的推理方案。此外，Ollama 与 [[Open WebUI]] 等前端界面的深度兼容，进一步提升了交互体验，使用户能够更高效地管理会话记录与模型配置，是构建现代化 AI 工作流的关键基础设施。

## 在本视频中的角色
在《Hermes Agent 高级玩法与优化指南简报》中，Ollama 被定位为 Hermes Agent 的核心运行平台。它主要承担了以下关键角色：
1. **模型运行载体**：作为 Hermes Agent 的内置运行环境，负责模型的加载与执行。
2. **资源优化工具**：通过云端接口调用功能，帮助用户规避本地资源瓶颈，实现免费模型的高效利用。
3. **部署简化器**：通过一键部署与标准化接口，降低了用户配置 Agent 的技术难度，是实现“主副模型”协作与 API 服务化的重要技术底座。
