# GPT-5.5

## 基本定位

GPT-5.5 是 OpenAI 继 GPT-4 之后推出的旗舰大语言模型，在本视频中被描述为“目前公认的最强模型之一”。它具备强大的推理、编程、多模态理解和指令遵循能力，是 Codex Chrome 插件的底层模型，负责驱动所有自动化任务的决策和执行。

## 核心特征/能力

1. **强推理与代码能力**：能够理解复杂的多步指令并生成高效代码，支持端到端的自动化工作流（如模拟用户点击、填写表单、数据验证）。
2. **多模态生成**：内置 GPT Image 2 图像生成能力，可生成高质量图片，且中文渲染完美。
3. **提示词优化**：能自动优化用户输入的简单描述，生成专业、详细的 Prompt，提升任务准确性。
4. **多代理协调**：支持分配子代理并协同调度，在 Codex Chrome 中实现多标签并行任务。
5. **环境适应能力**：能够检测站点安全策略并切换数据获取方式（如 DOM 渲染抓取），展现强大的抗干扰能力。
6. **快速推理模式**：提供 Medium/Fast 模式，在 Fast 模式下显著降低延迟，响应速度远超同类模型。

## 应用场景

1. **驱动 Codex Chrome 插件**：作为核心引擎，执行跨标签页的网页自动化任务（数据抓取、表单填写、社交发布）。
2. **内容创作**：自动生成营销文案、图片，并完成发布前的最终核查。
3. **数据分析与报告**：汇总大量 Web 内容，理解情绪，生成结构化表格或分析报告。
