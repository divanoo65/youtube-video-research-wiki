# Nous Research

## 基本定位

Nous Research 是一个以开源优先、去中心化为核心理念的 AI 研究团队，Hermes Agent 的开发者。除 Agent 框架外，团队还涉足分布式训练（DisTrO）和仿真环境研究（WorldSim）等前沿领域。

## 核心特征/能力

1. **开源优先理念**：所有核心项目（包括 Hermes Agent）以开源方式发布，推动社区共建。
2. **DisTrO 分布式训练**：研究去中心化的分布式训练技术，降低大模型训练对集中算力的依赖。
3. **仿真环境研究（WorldSim）**：探索 AI 在仿真环境中的学习与进化能力。
4. **Hermes Agent 开发**：以执行循环驱动架构和程序化知识生成为核心特色，挑战传统 Agent 框架设计范式。
5. **用户完全可控**：核心理念是用户对 Agent 拥有完全的控制权和数据自主权。

## 应用场景

- **开源 AI Agent 研发**：推动个人自托管 Agent 的技术演进和标准制定。
- **分布式训练研究**：降低大模型训练门槛，促进算力民主化。
- **仿真环境研究**：探索 AI 在虚拟环境中的自主学习与进化路径。
