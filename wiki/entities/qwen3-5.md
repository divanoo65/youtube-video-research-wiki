# Qwen3.5

## 基本定位

Qwen3.5 是阿里巴巴通义千问系列的开源模型，提供从 0.8B 到 9B 多种参数版本，专为显存受限的用户设计。在 Llama-cpp 引擎加持下，即使普通笔记本也能流畅运行，作为 Hermes Agent 的轻量级后端模型。

## 核心特征/能力

1. **多参数版本**：提供 0.8B、2B、4B、9B 等版本，适配不同显存配置。
2. **低显存需求**：最小版本可在 4G 显存设备上运行，极大降低硬件门槛。
3. **Llama-cpp 优化**：通过量化编译，在保持合理性能的同时减少显存占用。
4. **中文支持**：继承通义千问的中文理解能力。
5. **开源免费**：可自由部署，无需 API 费用。

## 应用场景

1. **低配笔记本部署**：作为入门级本地 AI 助手，满足日常问答和简单编程。
2. **移动端远程调用**：通过 Hermes Agent 和 Telegram 远程调用，执行轻量级任务。
3. **教育与学习**：用于学习 AI 模型部署和 Agent 开发。
