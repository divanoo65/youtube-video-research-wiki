## 基本定位

OpenAI 是全球领先的人工智能研究公司，以 GPT 系列大语言模型闻名。最新发布的 GPT-5.5 Instant 已作为 ChatGPT 默认模型向所有用户开放。

## 核心特征

- 开发了 GPT 系列旗舰模型
- ChatGPT 作为主要产品面向 C 端用户
- API 服务面向开发者，支持动态 `chat-latest` 调用
