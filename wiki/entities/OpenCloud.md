# OpenCloud

## 基本定位
OpenCloud 是一款开源的 AI Agent 工具，在功能定位上与 [[Hermes-Agent-赫美斯]] 构成直接竞争关系。根据 Hermes Agent 视频的描述，OpenCloud 的稳定性较差，更新时容易出现崩溃和引入 bug 的情况。

## 核心特征/能力
1. **定位相似**：与 Hermes Agent 同属本地 AI Agent 工具类别
2. **稳定性问题**：更新时容易崩溃或引入多种 bug
3. **开源社区**：作为开源项目拥有自己的用户社区

## 关系网络
