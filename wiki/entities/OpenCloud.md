# OpenCloud

## 基本定位

OpenCloud 是一个开源的 AI Agent 框架，与 [[Hermes-Agent]] 处于直接竞争关系。在社区讨论中，OpenCloud 虽然功能更新频繁，但每次更新都会引入大量 bug，导致系统不稳定，从而被用户诟病。

## 核心特征/能力

1. **更新频繁**: 社区持续发布新功能，迭代速度快。
2. **稳定性差**: 每次更新后容易出现崩溃或功能异常，需要用户频繁调试。
3. **GitHub 星标增速被超越**: 在同类框架中，Hermes Agent 的星标增长已超过 OpenCloud。

## 应用场景

1. **作为对比参照**: 在本视频中被用作 Hermes Agent 稳定性的反面案例。

## 关系网络

- [[Hermes-Agent]] — **直接竞争**: 两者均定位 AI Agent 框架，但在稳定性和更新策略上形成鲜明对比。

## 关键事件/里程碑

- **GitHub 星标被 Hermes Agent 超越**: 标志着社区认可度的转移。
