# MiniMax M2.7

## 基本定位

MiniMax M2.7 是 MiniMax 推出的云端免费模型，通过 Ollama 可直接调用，无需本地 GPU 资源。

## 核心特征/能力

1. **云端免费**：无需本地 GPU 资源，节省硬件成本。
2. **易用性**：通过 Ollama 即可直接调用，操作简便。
3. **兼容性好**：支持与 Hermes Agent 和 Ollama 集成，提供便捷的部署方式。
4. **适用场景广**：适用于需要云端算力的场景，如远程办公和资源受限环境。
5. **性能稳定**：在云端环境下表现稳定，适合长期使用。

## 应用场景

1. **云端计算**：适用于需要云端算力的场景，如远程办公和资源受限环境。
2. **低成本部署**：适合预算有限的用户，通过云端模型降低成本。
3. **多平台支持**：支持多种操作系统，方便用户在不同设备上使用。
