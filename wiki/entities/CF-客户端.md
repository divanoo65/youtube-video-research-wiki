# CF 客户端

## 基本定位
CF 客户端是一款专用的图形化桌面应用程序（约 130MB），用于在本地计算机上方便地加载和运行 AI 模型（如 Sulphur 2）。它提供了从模型下载、环境配置到参数调节的全流程管理，支持 NVIDIA、AMD 显卡及 CPU 解码，并内置了 LTX2.3 等工作流模板。

## 核心特征/能力
1. **一键环境配置**：首次运行时自动下载必要的 Python 环境包及依赖，无需用户手动安装。
2. **多语言支持**：安装完成后支持中文、繁体中文、英文界面切换。
3. **硬件选择**：初始设置可根据用户硬件选择计算驱动（NVIDIA GPU / AMD GPU / CPU）。
4. **模型管理**：提供图形界面，允许用户在 `Models/checkpoints` 目录放置主模型，在 `Models/GGUF` 目录放置提词器。
5. **工作流模板**：内置 LTX2.3 等视频模型模板，一键加载，自动检测缺失的辅助模型并提示下载。
6. **参数调节**：直观调节主模型切换、视频长度、分辨率、加速器 (LoRA) 开关等。
7. **自动输出**：生成的视频自动保存到 `Documents/CF/Output/Video` 目录。

## 应用场景
- **本地模型快速部署**：用户无需编写代码，通过 CF 客户端即可完成从模型下载到生成视频的完整流程。
- **模型调试与测试**：开发者可快速切换不同精度版本（FP8/BF16），测试硬件兼容性和生成效果。
- **教学展示**：用于向非技术人员演示 AI 视频生成能力，降低使用门槛。
