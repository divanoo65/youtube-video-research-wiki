# OpenAI

## 基本定位

OpenAI 是人工智能研究公司，以开发 GPT 系列大语言模型闻名。GPT-5.5 Instant 是其最新默认模型，免费向所有用户开放，标志着公司战略从参数竞赛转向实用性和安全性。

## 核心特征/能力

1. **GPT 系列开发**：持续迭代 GPT 模型，从 GPT-3 到 GPT-5.5。
2. **API 服务**：提供商业 API，支持 `chat-latest` 动态调用。
3. **免费开放策略**：GPT-5.5 Instant 向所有免费用户开放。
4. **安全研究**：注重模型安全，减少幻觉和有害输出。

## 应用场景

1. **个人助手**：通过 ChatGPT 提供日常问答、创作辅助。
2. **开发者工具**：通过 API 集成到第三方应用。
