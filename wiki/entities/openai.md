## 基本定位

OpenAI 是全球领先的人工智能研究实验室，以开发 GPT 系列大语言模型和 ChatGPT 闻名。近年来转向免费策略以扩大用户基础。

## 核心特征/能力

1. **模型开发**：迭代开发 GPT 系列，从 GPT-3 到 GPT-5.5 Instant。
2. **ChatGPT 平台**：全球最流行的 AI 对话产品，集成搜索、图像识别等能力。
3. **学术研究**：发表多项重要 AI 论文，推动 RLHF 等技术。
4. **商业模式**：由非盈利转向商业化，推出 API 服务。
5. **政策变化**：2026 年将最强模型免费开放，大幅降低使用门槛。

## 应用场景

- **日常问答**：ChatGPT 用户直接使用最新模型。
- **开发者集成**：通过 API 调用 GPT-5.5 Instant 开发各类应用。
