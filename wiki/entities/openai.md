# OpenAI

## 基本定位

全球领先的人工智能研究和部署机构，开发了 GPT 系列大语言模型和 ChatGPT 对话平台。

## 核心特征

1. **GPT 系列模型**：从 GPT-3、GPT-4 到 GPT-5.5 Instant，持续迭代最前沿的大语言模型
2. **ChatGPT 平台**：面向消费者的旗舰 AI 对话产品，免费+付费多层订阅
3. **开发者 API**：提供统一的 API 接口（如 `chat-latest`），持续降低开发者集成成本
4. **多模态能力**：支持文本、图像理解、联网搜索等多模态交互
5. **用户规模**：产品和模型面向全球用户开放，免费用户也可使用最新模型

## 关系网络

- 开发了 [[gpt-5-5-instant]] 模型
- 运营 [[chatgpt]] 对话平台
- API 简化策略影响开发者生态
- 与本地部署方案形成对比（参见 [[零成本部署]]）
