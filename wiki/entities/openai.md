# OpenAI

## 基本定位

OpenAI 是全球领先的人工智能研究公司，致力于开发安全、有益的通用人工智能。

## 核心特征

- 发布了 GPT 系列大语言模型，包括最新的 **GPT-5.5 Instant**
- 引入 `ch-latest` API 别名机制，简化开发者生态维护
- 推动模型产品从付费专享向全面免费开放转型
- 关注模型在实际应用中的可靠性，重点优化幻觉抑制
- 通过 Memory Sources 功能推进 AI 从工具向"长期陪伴助手"进化

## 关系网络

- 开发了 [[gpt-5.5-instant]] 模型
- 与 [[hermes-agent]] 同为 AI 工具生态核心玩家（[[hermes-agent]] 支持调用 OpenAI API）
