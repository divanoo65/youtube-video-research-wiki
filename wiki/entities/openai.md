# OpenAI

## 基本定位

OpenAI 是全球领先的人工智能研究机构，发布了 GPT-5.5 Instant 等先进的大语言模型。

## 与 GPT-5.5 Instant 的关系

- OpenAI 发布了 GPT-5.5 Instant 作为其最新、最强的生成式 AI 模型
- 引入 `chat-latest` API 调用策略，简化开发者对接流程
- 该模型已替换上一代模型成为 ChatGPT 默认模型
