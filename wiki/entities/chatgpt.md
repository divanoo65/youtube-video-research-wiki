## 基本定位

ChatGPT 是 OpenAI 推出的旗舰对话式 AI 产品，集成多模态、记忆、联网搜索等能力。2026年5月起，GPT-5.5 Instant 成为其默认模型。

## 核心特征/能力

1. **默认模型升级**：GPT-5.5 Instant 作为默认模型，免费用户也可使用。
2. **多功能集成**：除对话外，支持图片分析、文件上传、网络搜索。
3. **记忆功能**：可参考过去聊天记录提供个性化建议。
4. **跨平台**：Web、移动端、桌面端均可使用。
5. **API 支持**：开发者可通过 API 调用其能力。

## 应用场景

- **日常问答**：用户直接通过 ChatGPT 使用最新模型。
- **内容生成**：视频大纲、文章写作等。
