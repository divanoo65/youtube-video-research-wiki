# Codex Chrome

## 基本定位

Codex Chrome 是 OpenAI 开发的一款 **Chrome 浏览器插件**，核心功能是让 AI 代理在浏览器后台自主运行，支持多标签页并行处理复杂任务。它基于 GPT-5.5 模型，利用 Chrome DevTools Protocol 无缝操作已登录的浏览器环境，实现从信息检索到内容发布的全链路自动化。

## 核心特征/能力

1. **多标签并行运行**：AI 代理可以在独立标签页中单独运行，用户同时处理其他任务互不干扰；支持同时启动多个代理（如四个代理并行加入游戏房间并绘图）。
2. **子代理协作**：在复杂任务中可分配多个子代理协同操作，每个子代理负责独立子任务，最终汇总结果。
3. **DOM 渲染抓取**：当服务器端抓取被防火墙拦截时，通过 Chrome 渲染后的 DOM 树获取数据，有效绕过站点防护。
4. **提示词自动优化**：AI 能自主改进用户输入的简单指令，生成更规范、更有效的 Prompt，提升任务执行质量。
5. **多种思考模式**：提供“Medium（中等）”和“Fast（快速）”模式，用户可根据任务复杂度和速度需求切换。
6. **文件系统访问**：支持读取本地文件（如 PDF 收据、图片），需在插件设置中手动开启“允许访问文件网址”。
7. **跨平台扩展**：未来将推出手机 App，支持移动端远程下发指令。

## 应用场景

1. **财务报销自动化**：从 Gmail 筛选特定邮件，匹配本地收据 PDF（日期、金额、商户），自动登录报销系统填写表单、选择分类、上传附件并提交。
2. **社交媒体内容发布**：搜索信息 → 创作文案 → 生成 AI 图片（GPT Image 2） → 自动登录平台并发布，形成完整闭环。
3. **开发者社区数据采集**：抓取一周内热帖，分析用户情绪与关键问题，导出经过验证的 Excel 表格。
