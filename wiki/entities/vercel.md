Vercel 是一家领先的云平台公司，专注于为前端开发人员提供高性能、可扩展的网站和 Web 应用程序部署服务。它以其零配置部署、全球 CDN、自动 SSL 证书以及与流行前端框架（如 React、Next.js、Vue 等）的深度集成而闻名。Vercel 的核心理念是简化开发到部署的流程，让开发者能够专注于代码本身，而无需过多关注底层基础设施的配置和维护。通过提供预览部署、即时回滚和边缘函数等功能，Vercel 极大地提升了现代 Web 开发的效率和体验。它已经成为许多初创公司和大型企业部署其前端应用的首选平台之一，尤其是在 Jamstack 架构日益普及的背景下。

### 在本视频中的角色
在[[简报：从“氛围编程”到“智能体工程”——安德烈·卡帕斯（Andrej Karpathy）的技术洞察]]中，Vercel 被用作当前“人类优先”基础设施的一个典型例子。报告指出，现有的部署流程，例如 Vercel 的配置和 [[DNS]] 设置，主要是为人类用户设计的，需要人类进行手动操作和理解。卡帕斯以此为例，提出了对未来基础设施的展望，即应提供更简洁的“传感器”和“执行器”接口，以便[[智能体]]能够直接进行端到端的应用构建与部署，从而实现“智能体优先”（Agent-First）的范式转变。Vercel 在此语境下代表了当前高效但仍需人类干预的部署现状，与报告中设想的自动化、智能体驱动的未来形成了鲜明对比。
