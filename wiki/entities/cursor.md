# Cursor

Cursor 是一款基于 AI 驱动的现代代码编辑器，旨在通过深度集成人工智能技术来提升开发者的编程效率。它不仅具备传统 IDE 的代码补全、调试和版本控制功能，更通过内置的 AI 模型（如 Claude 等）实现了代码生成、重构、解释及错误修复等高级辅助能力。Cursor 的设计理念在于将 AI 深度嵌入开发工作流，使开发者能够通过自然语言与代码库进行交互，从而显著降低编码门槛并加速开发周期。

### 在本视频中的角色

在 [[Hermes Agent 智能中控台：全方位可视化管理与性能监测简报]] 所描述的系统架构中，Cursor 扮演着“被监控进程”的角色。作为 Hermes Agent 智能中控台实时捕获的 12 个“Alive Agent”（活跃代理/进程）之一，Cursor 的运行状态、资源占用情况以及与系统的交互行为均被纳入中控台的统一可视化管理范畴。这表明 Hermes Agent 具备跨工具的进程监控能力，能够将 Cursor 视为其生态系统中的一个关键组件进行性能监测与调度管理。
