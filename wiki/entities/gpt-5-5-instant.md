# GPT-5.5 Instant

## 基本定位

OpenAI 发布的最新、最强生成式 AI 模型，已替换上一代成为 ChatGPT 默认模型，对免费用户全面开放。

## 核心特征

1. **全员覆盖：** 对免费、Plus、Team 用户全面开放，无需订阅即可使用
2. **API 简化：** 支持 `chat-latest` API 策略，自动连接最新模型架构
3. **自我纠错：** 能识别数学推导中的中间步骤错误并重新推导
4. **对话自然化：** 减少过度礼貌的废话和奉承话，回答更加直接
5. **Memory Sources：** 能参考过去聊天记录，提供个性化建议
6. **多模态增强：** 在图片分析、文件理解和图表解读方面更加稳定
