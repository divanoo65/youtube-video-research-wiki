# Gemini 2.0 Flash

## 基本定位

Gemini 2.0 Flash 是 Google 推出的轻量级高性能模型，被 Hermes Agent 推荐作为副模型（辅助模型）以降低 Token 消耗。

## 核心特征/能力

1. **轻量级**：模型体积小，适合用于副模型任务。
2. **高性能**：在处理简单任务时表现优异，能够有效降低 Token 消耗。
3. **兼容性好**：支持与 Hermes Agent 集成，作为副模型使用。
4. **适用场景广**：适用于审批、压缩、MCP 调用、会话搜索、技能检索、网页/视觉相关任务等。
5. **成本效益高**：性价比高，适合预算有限的用户。

## 应用场景

1. **副模型任务**：适用于需要低消耗的辅助任务，如审批和压缩。
2. **多任务处理**：适合需要同时处理多个任务的场景。
3. **成本控制**：适合预算有限的用户，通过副模型降低整体成本。
