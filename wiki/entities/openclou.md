# OpenCloud

## 基本定位
OpenCloud（中文名：小龙虾）是一个与 Hermes Agent 同类的 AI Agent 项目。根据视频中的对比，OpenCloud 在稳定性方面表现较差，每次更新都会引入较多 Bug，导致系统崩溃。

## 核心特征/能力

1. **同类项目**: 与 Hermes Agent 同属 AI Agent 框架类别。
2. **稳定性较差**: 每次更新都会引入非常多的 Bug，导致系统崩溃。
3. **GitHub 增长较慢**: 其 GitHub 仓库的 Star 增长速度已被 Hermes Agent 超越。
