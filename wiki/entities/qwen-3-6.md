# Qwen 3.6

## 基本定位

Qwen 3.6 是阿里巴巴通义千问（Qwen）系列的最新一代开源大语言模型，提供从 0.8B 到 27B 多种参数规模，覆盖不同显存需求的硬件配置。在本地部署场景中，Qwen 3.6 与 Hermes Agent 组合使用，通过 Llama-cpp 推理后端实现零月费、零 Token 成本的私有 AI 助手。

## 核心特征/能力

1. **多种参数规模**：提供 0.8B、2B、4B、9B、27B 等不同规模的模型，适配从低端到高端的各类硬件配置。
2. **中文理解优秀**：在中文逻辑推理、代码编写等任务上表现卓越，适合中文用户的本地部署场景。
3. **代码生成能力**：在实测中能够编写带前端、后台及数据库的完整 PHP 程序。
4. **Llama-cpp 兼容**：支持通过 Llama-cpp 推理框架运行，能有效防止显存溢出（OOM），适合显存容量有限的个人电脑。
5. **可关闭深度思考模式**：在对接 Agent 时，关闭深度思考模式可显著提升响应速度（实测 39.51 Token/秒）。
6. **国内镜像可用**：支持通过国内镜像 ModelScope 下载，解决网络访问问题。

## 应用场景

- **零成本本地 AI 助手**：搭配 Hermes Agent 构建 24 小时在线的私有 AI 助手，无需云 API 费用。
- **代码编程辅助**：用于 PHP 程序编写、UI 设计等代码生成任务。
- **中文推理任务**：处理需要中文语境理解的分析、总结和推理工作。
