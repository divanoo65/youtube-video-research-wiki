    12|## 基本定位
    13|
    14|Qwen 3.6 是阿里巴巴开发的大语言模型系列，以强大的中文能力、逻辑推理和代码编写为特点，通过 Llama-cpp 可在本地部署，适合作为 Hermes Agent 的底层模型。
    15|
    16|## 核心特征/能力
    17|
    18|1. **多尺寸可选**：提供 0.8B、2B、4B、9B、27B 等多个版本，适配从低端显卡到高性能工作站的硬件环境。24G 显存推荐 27B 版本（约 17G 大小）。
    19|2. **高性能推理**：在 27B 版本上，未优化时可达约 40 Tokens/s 的生成速度，优化后（如编译优化、关闭深度思考）可达 50-60 Tokens/s。
    20|3. **深度思考模式**：支持常规快速响应和深度思考两种推理模式，可通过参数关闭以提升任务响应速度。
    21|4. **中文理解领先**：在中文本地化理解、语义分析任务上表现优于同尺寸的其他开源模型。
    22|5. **代码生成能力**：能够生成复杂 Web 应用架构（如 PHP+数据库的中转站），包括前端 UI 设计和后台逻辑统计。
    23|6. **本地部署兼容**：通过 Llama-cpp 运行，支持 GPU 加速（CUDA），无需联网，保障数据隐私。
    24|
    25|## 应用场景
    26|
    27|1. **开发辅助**：在本地生成代码片段、调试脚本、设计 API 接口，全部在本地完成，不泄露代码。
    28|2. **自动化任务**：配合 Hermes Agent 执行定时任务，如每日数据汇总、文件整理等。
    29|3. **远程查询**：通过 Telegram Bot 在手机上询问问题，利用本地算力快速获取答案。
