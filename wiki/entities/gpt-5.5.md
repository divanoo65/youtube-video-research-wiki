# GPT-5.5

## 基本定位

GPT-5.5 是由 OpenAI 发布的最新版本大语言模型，在 Codex Chrome 插件中担任核心推理引擎。它专为计算机使用（Computer Use）场景优化，在保持深度推理能力的同时提供极快的响应速度。

## 核心特征/能力

1. **双模式运行**：提供 Fast（快速）与 Medium（中等思考深度）两种模式。Fast 模式下处理速度远超以往任何 AI Agent，Medium 模式更适合复杂逻辑分析。
2. **深度推理**：能够理解多步骤业务逻辑（如报销流程中的邮件检索→PDF匹配→表单填写），并自行规划执行路径。
3. **高效文本理解**：对长文档、多语言内容（中英文混合）具有优秀理解力，可精准提取关键信息。
4. **Agent 决策能力**：能自主判断何时切换搜索关键词、何时滚动页面、何时打开链接阅读全文。
5. **多工具调用**：在插件内部可调用文件系统、浏览器 DOM、外部 API 等多种工具。

## 应用场景

- **提示词优化**：实测中 Codex 登录 ChatGPT 官网，让 GPT-5.5 优化自身提示词，生成极其规范的指令。
- **数据提取与总结**：从开发者社区抓取 20+ 帖子后自动总结重点与表格，耗时仅 3 分 41 秒。
- **多代理协作**：支持同时驱动四个子代理并行执行绘图任务，每个代理独立调用 GPT-5.5 进行推理。
