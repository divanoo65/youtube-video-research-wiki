在 [[Hermes Agent 智能中控台：全方位可视化管理简报]]中，[[User Profile]]（用户画像）是该智能中控台“记忆管理与纠错机制”模块中的一个关键组成部分。它与 [[Agent Memory]]（工作记忆）一同被分类显示，旨在将通常不可见的 [[LLM 记忆]]转化为可交互、可管理的数据。用户画像允许用户直接在网页端对存储的用户相关信息进行增加、删除和修改操作，极大地简化了传统上需要手动翻阅本地 Markdown 文件来管理这些数据的繁琐过程。这一功能体现了中控台在[[数据透明化]]和[[记忆可编辑化]]方面的核心优势，使用户能够更直观、高效地管理和优化AI的个性化行为与响应。通过对用户画像的直接编辑，系统能够更好地理解和适应用户的偏好与习惯，从而提升AI服务的质量和用户体验。

**在本视频中的角色：**
[[User Profile]] 在视频中作为 [[Hermes Agent 智能中控台：全方位可视化管理简报]]的核心竞争优势之一——“记忆管理与纠错机制”模块下的一个重要功能被介绍。它与 [[Agent Memory]] 并列，共同实现了对AI内部记忆的可视化和编辑，使用户能够直接在网页端管理和修改用户画像数据，从而影响AI的行为和响应，无需手动操作本地文件。
