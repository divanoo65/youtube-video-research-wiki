## 操作指南与实践洞察

### 1. 快速部署流程
要使用此中控台，前提是计算机上已安装并运行过 Hermes Agent 本体。
*   **克隆项目：** 使用 Git 克隆中控台仓库。
*   **自动化安装：** 运行 `install` 指令，系统将自动创建虚拟环境、安装套件并构建前端（Build frontend）。
*   **启动：** 安装完成后点击生成的链接，即可进入带有开机动画的控制界面。

### 2. 行为模式分析（Pattern Insight）
*   **任务聚类：** 系统会自动将所有会话分成六大类，帮助用户了解自己最常要求 AI 执行的任务。
*   **自动化建议：** 当系统检测到某个 Prompt 重复出现 3 次以上时，会标记“闪电”图标，提示用户应考虑将其转化为自动化脚本（Skill）。

### 3. 高级交互技巧
*   **全文本搜索：** 支持在数百个会话中同时检索标题和内文，并明确标识关键词所在位置。
*   **即时任务干预：** 在 Cron 标签页中，用户可以随时对后台任务执行恢复（Resume）或删除操作。
