## 图谱（Obsidian 预览中打开）

```mermaid
mindmap
root((Hermes Agent AI 中控台 （Dashboard）))
  核心功能模块
    Dashboard （仪表盘）
      Overview: 模型与后端状态
      AI 成绩单: 对话/消息/工具调用数
      Token 统计: 累计处理量
      Growth Delta: 每日增长快照
      工具使用排行与活动折线图
    Memory & Sessions （记忆与会话）
      Memory Tab: 编辑/删除/新增记忆
      User Profile: 用户画像管理
      Session Step: 完整对话记录查看
      全文搜索: 标题与内文检索
    Cost & Health （成本与健康）
      Cost Tab: 按模型拆分费用明细
      Health Tab: API Key 与服务状态监控
    Skills & Tasks （技能与任务）
      Skills Tab: 80个技能分类管理
      Cron Tab: 后台排程任务管理
      Project Step: 自动侦测项目代码状态
  进阶分析与交互
    Pattern （习惯分析）
      Task Clusters: 任务自动分类
      24小时活动图
      重复 Prompt 闪电提醒 （自动化建议）
    Correction & Chat （纠错与对话）
      Correction Tab: AI 错题本
      Wchat Tab: 浏览器直接调用 CLI 对话
  安装与配置
    前提: 需先运行 Hermes Agent 本体
    步骤: Git Clone 项目
    步骤: 执行 Install 指令自动构建
    交互: 快捷键 Ctrl+K 搜索 / T键切换主题
```
