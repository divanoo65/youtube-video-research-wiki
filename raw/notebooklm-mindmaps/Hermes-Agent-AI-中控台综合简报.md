## 图谱（Obsidian 预览中打开）

```mermaid
mindmap
root((Hermes Agent AI 中控台 （H-Dashboard）))
  Dashboard （仪表盘）
    Overview: 模型、后端与运行时间
    大脑状态: state.db 数据库大小
    成绩单: 对话数、消息数、工具调用、技能数
    平台分布: CLI 与 Telegram 统计
    Token 消耗总量
    存储空间: 记忆与用户画像占比
    成长曲线: 每日快照与增长趋势
    排行榜: 常用工具与活动折线图
  Memory & Sessions （记忆与会话）
    Memory Tab: 工作记忆与用户画像
    记忆编辑: 自动分类、编辑与双重确认删除
    Session Steps: 对话记录列表
    全文搜索: 同时搜索标题与内文
    详细内容: 查看完整对话与自动化任务
  Cost & Health （成本与健康度）
    Cost Tab: 今日费用与累计费用
    模型拆分: 各模型费用占比 （如 Opus）
    Health Tab: API Key 状态检查
    服务状态: Telegram Gateway 运行状态
  Skill & Tasks （技能与任务）
    Skills Tab: 80个技能分类与自定义技能
    Cron Tab: 排程任务管理 （摘要、检查、备份）
    Project Step: 自动侦测项目、Git 状态与语言
  Advanced Analysis （深度分析）
    Pattern: 使用习惯与任务聚类
    活动分析: 24小时高峰期图表
    自动化建议: 闪电标识的高频 Prompt
    Agents 监控: 识别电脑运行中的所有 AI Agent
    Correction Tab: 错题本 （错误记录与纠正）
  Interactive & Setup （交互与安装）
    Chat Tab: 浏览器直接调用 Hermes 聊
    快捷键: T键切换主题, Ctrl+K 搜索跳转
    安装步骤: 克隆项目、运行 install、环境构建
```
