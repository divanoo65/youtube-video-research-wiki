## 图谱（Obsidian 预览中打开）

```mermaid
mindmap
root((Hermes Agent AI 中控台 （Dashboard）))
  Dashboard （总览仪表盘）
    Overview: 模型、后端与运行天数
    state.db: AI 学习数据量/大脑大小
    成绩单: 对话数、消息量、工具调用、技能数
    Token 消耗统计 （128万）
    Growth Delta: 每日快照与成长曲线
  Memory & Sessions （记忆与会话）
    Memory Tab: 工作记忆与用户画像
    记忆编辑: 直接在线修改/删除/添加
    Session Steps: 完整对话记录与搜索
    全文明细: 支持标题与内文关键字查找
  Cost & System （成本与系统）
    Cost Tab: 每日/累计费用计算
    模型拆分: 不同模型 （如 Opus） 费用名细
    Health Tab: API Key 与服务运行状态
  Analysis & Skills （分析与技能）
    Pattern: 任务分类与活跃时间分析
    闪电标记: 识别高频 Prompt 并建议自动化
    Skills Tab: 80个技能分类与自定义技能管理
  Automation & Projects （自动化与项目）
    Cron Tab: 后台排程任务管理 （摘要、扫描、备份）
    Project Steps: 自动侦测项目状态与 Git 提交情况
    Agent 监控: 识别电脑上运行的所有 AI Agent
  Correction & Chat （纠错与交互）
    Correction Tab: AI 错题本 （关键/重要/次要纠正）
    Web Chat: 直接在浏览器与 Hermes 对话
    快捷功能: T 键切换主题, Ctrl+K 搜索跳转
  安装与配置
    前提: 需先安装并运行 Hermes Agent 本体
    步骤: Git Clone 项目 -> install 指令 -> 启动链接
```
