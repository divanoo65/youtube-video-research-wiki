## 图谱（Obsidian 预览中打开）

```mermaid
mindmap
root((Hermes Agent AI 中控台 （HOD）))
  Dashboard （仪表盘）
    Overview: 模型与后端状态
    AI 成绩单: 对话、消息、工具调用数
    Token 处理量与费用统计
    Growth Delta: 每日增长快照
    活动量折线图与常用工具榜
  Memory & Sessions （记忆与会话）
    Memory Tab: 编辑/删除工作记忆与用户画像
    Session Step: 完整对话记录查看
    全文搜索: 标题与内文关键字检索
    自动分类: 标注 Telegram 或 CLI 来源
  Management （管理功能）
    Cost Tab: 按模型拆分费用明细
    Health Tab: API Key 与服务运行状态
    Skills Tab: 80个技能分类与自定义技能管
    Chrome Tab: 背景自动排程任务管理
    Project Step: 自动侦测 Git 项目与代码状态
  Analysis （行为分析）
    Pattern: 任务聚类与 24 小时活动图
    重复 Prompt 识别: 自动提示转换为自动化
    Correction Tab: 错题本（纠正记录）
  Interaction （交互体验）
    Chat Tab: 浏览器内直接对话
    Theme Picker: 快捷键 T 切换主题
    Quick Jump: Ctrl+K 快速搜索标签页
  Installation （安装配置）
    前提: 需先安装 Hermes Agent 本体
    步骤: 克隆项目 -> install 指令
    自动化: 自动构建虚拟环境与前端
```
