- [[entities/Hermes Agent|Hermes Agent]] — 由 Nous Research 开发的开源 AI Agent 框架，采用执行循环架构，支持自我进化、程序化知识生成和四层记忆体系。
- [[entities/Nous Research|Nous Research]] — Hermes Agent 的研发团队，专注于开源 AI Agent 框架和分布式训练技术的前沿研究机构。
- [[entities/OpenAI|OpenAI]] — 全球领先的人工智能研究机构，GPT-5.5 Instant、ChatGPT 等产品的开发者，在 LLM 领域保持技术前沿地位。
- [[entities/OpenClaw|OpenClaw]] — 自托管 AI Agent 平台，采用中心化 Gateway 网关架构，以高稳定性、可审计性和透明性著称。

## Concepts
- [[concepts/Agent 执行循环|Agent 执行循环]] — Hermes Agent 的核心编排机制，以 AI Agent 自身的执行→学习→改进闭环驱动整个系统，取代传统的中心化网关架构。
- [[concepts/Llama-cpp 推理框架|Llama-cpp 推理框架]] — 高性能本地 LLM 推理引擎，通过 C++ 实现和 CUDA 加速在消费级显卡上高效运行各种开源模型。
- [[concepts/Memory Sources|Memory Sources]] — GPT-5.5 Instant 引入的长效记忆功能，使模型能参考过去聊天记录和关联数据，演变为个性化 AI 助理。
- [[concepts/WSL 部署|WSL 部署]] — Windows Subsystem for Linux (WSL) 在 Windows 系统上运行 Linux 环境的技术方案，是本地 AI 部署的重要基础设施。
- [[concepts/chat-latest API 策略|chat-latest API 策略]] — OpenAI 为 GPT-5.5 Instant 推出的统一 API 端点策略，开发者只需调用固定端点名称即可自动获取最新模型。
- [[concepts/五层纵深防御模型|五层纵深防御模型]] — Hermes Agent 针对本地智能体高权限风险设计的多层安全防护体系，涵盖用户授权、命令审批、容器隔离、凭证过滤和上下文扫描。
- [[concepts/分层记忆体系|分层记忆体系]] — Hermes Agent 的四层结构化记忆系统，包含核心持久记忆、会话历史、扩展画像和程序性技能记忆，通过按需检索实现高效上下文管理。
- [[concepts/多模态 AI|多模态 AI]] — AI 模型同时处理和理解文本、图像、图表等多类型输入的能力，是 GPT-5.5 Instant 的核心增强特性之一。
- [[concepts/幻觉率控制|幻觉率控制]] — LLM 在生成内容时避免产生虚假、错误信息的机制，是 GPT-5.5 Instant 等新一代模型的核心优化方向。
- [[concepts/本地 AI Agent 部署|本地 AI Agent 部署]] — 在本地硬件上部署开源 AI 模型的完整方法论，涵盖环境搭建、模型选择、Agent 配置和远程接入，实现零成本的私有 AI 服务。
- [[concepts/程序化知识|程序化知识]] — Agent 能根据执行经验自动生成可复用技能的知识形态，标志着智能体从『记住事实』升级为『记住方法』。
