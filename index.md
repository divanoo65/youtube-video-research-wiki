## Entities
- [[entities/Codex-Chrome插件|Codex Chrome 插件]] — OpenAI 推出的 Chrome 浏览器插件，集成 GPT-5.5 模型，实现多标签并行、后台静默的计算机使用，支持端到端自动化办公、深度数据提取和多代理协作。
- [[entities/gpt-5.5|GPT-5.5]] — OpenAI 开发的最新大语言模型，集成于 Codex Chrome 插件，提供 Fast 与 Medium 两种推理模式，支持复杂任务深度思考与快速执行。
- [[entities/Chrome调试协议|Chrome 调试协议（CDP）]] — Chrome 浏览器暴露的调试协议，允许外部工具通过 WebSocket 连接控制浏览器的标签页、DOM、网络请求、输入模拟等底层功能。
- [[entities/模型上下文协议|模型上下文协议（MCP）]] — 允许 AI 代理连接外部客户端应用程序的协议，可绕过网页端安全策略拦截，是 Codex 插件在受限站点的替代方案。

## Concepts
- [[concepts/多标签并行处理]] — Codex 插件允许 AI 在后台创建多个独立标签页并行执行任务，互不干扰。
- [[concepts/后台静默运行]] — AI 代理在不干扰用户当前浏览器操作的情况下在后台标签页独立执行任务的能力。
- [[concepts/计算机使用]] — AI 代理直接操控计算机图形用户界面执行操作的能力，而非仅通过文本或 API 交互。
- [[concepts/DOM渲染数据获取]] — 通过浏览器渲染后的 DOM 获取数据，绕过服务器端反爬机制。
- [[concepts/MCP模型上下文协议]] — 允许 AI 代理通过专用客户端访问受限制平台的协议。
- [[concepts/多代理协作]] — 多个独立的 AI 子代理在各自隔离的浏览器标签页中同时工作，协同完成复杂任务。
- [[concepts/端到端自动化]] — AI 代理从起始输入到最终输出完成整套业务流程，无需人工干预。
