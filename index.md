## Concepts

- **Token自由** — 通过本地部署开源模型，实现零 Token 费用、无调用限制的自由使用状态。
- **chat-latest-API** — OpenAI 引入的统一 API 调用策略，开发者只需调用此名称即可自动连接到最新模型架构。
- **五层纵深防御** — Hermes Agent 的默认安全模型，通过多层防护机制防范本地智能体的高权限风险。
- **分层记忆体系** — Hermes Agent 的四层存储架构，平衡了 Token 效率与长期召回能力。
- **幻觉率** — 大语言模型在回答中产生虚构或错误信息的比率，是衡量 AI 可靠性的核心指标。
- **执行循环驱动架构** — Hermes Agent 的核心架构模式，以智能体自身的执行循环为编排引擎的设计范式。
- **程序化知识生成** — 智能体从执行经验中自动提取并固化可复用操作流程的能力，实现从"记住事实"到"记住方法"的跨越。
- **自我纠错机制** — 大语言模型在推理过程中主动识别并修正自身或用户错误的能力，是提升模型可靠性的关键技术路径。
