## Concepts
- [[cron计划任务]] — Hermes Agent 内置的定时任务系统，支持自然语言设定计划。
- [[ch-latest接口]] — OpenAI 推出的统一 API 名称，自动获取最新 Instant 模型架构。
- [[程序化知识]] — 由 AI 智能体基于执行经验自动生成的可复用工作流。
- [[分层记忆体系]] — Hermes Agent 的四层结构化记忆系统：核心持久、会话历史、用户画像、程序性记忆。
- [[gguf格式]] — 存储量化大语言模型权重的文件格式，由 Llama-cpp 推广。
- [[mcp协议]] — Model Context Protocol，智能体与外部工具交互的标准化协议。
- [[深度思考模式]] — Qwen 3.6 的推理增强模式，逻辑严密但响应稍慢。
- [[推理引擎]] — 运行大语言模型并生成输出的后端软件，Llama-cpp 是推荐。
- [[五层纵深防御]] — Hermes Agent 的多层安全模型，从授权到容器隔离。
- [[wsl2]] — Windows Subsystem for Linux 2，在 Windows 上运行 Linux 内核的虚拟化技术。
- [[执行循环]] — Hermes Agent 的核心编排机制，将 AI 的“感知-计划-行动-评估”循环作为引擎。
- [[自我纠错机制]] — 模型检测并修正自身逻辑错误的能力，GPT-5.5 Instant 的核心突破。
- [[记忆与个性化]] — 模型持久化用户偏好并提供个性化响应的能力。
